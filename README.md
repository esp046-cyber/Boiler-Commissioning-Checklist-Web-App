# Boiler-Commissioning-Checklist-Web-App
To ensure all steps in boiler electrical commissioning are followed properly and documented.

 Features:

 Interactive step-by-step checklist
 Sections for:

   PLC/HMI program editing steps
   I/O termination verification
   VFD parameter setup
   Instrument loop-check
   Simulation & FAT/SAT procedures
 Photo upload for cable terminations
 E-signature for completed tasks
 Export to PDF for reporting

 Why it helps:

Prevents missed steps, and gives a ready report for the client
