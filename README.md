## How to Use

### 1. Project Setup
1. Fill in project information (Project Name, Boiler Model, Technician Name, Date)
2. All information is automatically saved

### 2. Navigate Sections
1. Use the tab navigation to move between sections
2. Each section focuses on a specific commissioning area
3. Progress is tracked separately for each section

### 3. Complete Tasks
1. Tap on any task to mark it as complete
2. Completed tasks show a green border and strikethrough
3. Action buttons become available after task completion

### 4. Add Documentation
1. **Photos**: Tap camera icon to upload/take photos
2. **Notes**: Tap notes icon to add detailed observations
3. **Signature**: Tap signature icon to digitally sign completed tasks

### 5. Track Progress
1. Monitor progress in the header section
2. Progress bar shows completion percentage
3. Export becomes available when all sections are complete

### 6. Export Report
1. Click "Export PDF" button when all sections complete
2. Professional report generated with all documentation
3. Report includes photos, signatures, notes, and progress summary

## Installation & Usage

### Local Deployment
1. Download all files (index.html, styles.css, app.js)
2. Open index.html in a modern web browser
3. No server required - works completely offline

### Recommended Setup
1. Add to home screen on mobile devices for app-like experience
2. Use in landscape mode on tablets for optimal viewing
3. Ensure adequate lighting for photo capture
4. Test signature functionality before field use

## Data Privacy & Security
- All data stored locally on device
- No data transmitted to external servers
- Photos and signatures remain on device
- Export data as JSON for secure backup

## Troubleshooting

### Photo Upload Issues
- Ensure camera permissions granted
- Try different file formats (JPEG, PNG)
- Clear browser cache if problems persist

### Signature Problems
- Use finger/stylus for best results on touch devices
- Clear signature and try again if canvas unresponsive
- Ensure name field filled before confirming

### Export Failures
- Check all sections are 100% complete
- Verify project information filled
- Try different browser if PDF generation fails
